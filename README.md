# Mojave 离线动态壁纸

简介：基于 Mojave 动态壁纸的离线增强版，支持固定地点离线计算日出日落，并以本地时间驱动切换。

## 致谢
本项目基于 https://github.com/Yaindrop/DynamicWallpaper ，所有工作都在其基础上完成，特此感谢原作者的开源贡献。

## 我做的改动
- 改为全离线逻辑，移除联网定位与外部日出日落接口依赖。
- 新增“固定地点（离线）”配置：通过固定经纬度离线计算日出日落。
- 日出日落不可用时回退到手动时间或均分 24 小时的时间轴。
- 优化时间段判断，避免跨午夜时段计算错误。
- 修复预加载方式，避免无效缓存。
- 过渡动画仅作用于 `background-image`，避免不必要的全量过渡。

## 使用说明（简要）
- 在属性中开启“固定地点（离线）”。
- 通过滑条设置经纬度（×100），例如成都：纬度 3057，经度 10407。
